What's new in 418

Refresh your Add Share Do browser window to get the new features just released as verson 418:

  • Tags can now have descriptions.  When you share a tag with someone we email them an invitation and now include the description you have given the tag.  It provides more context.  To set a description click the "edit" button next to the tag name, fill in the field, and press "update".  Change your mind as often as you like.  Click the "edit" link again to close the edit window.
  • Add Share Do is not for everyone: you'll find the "close account" link on the Preferences page.
  • Improved reliability for tasks sent in via email. Before this release it was possible for tasks emailed in to be held up for a day or two if we lost connection to our email server. The longest time now is 30 minutes. 

If you're new to Add Share Do, don't forget to sign up and try it for free.
